# 📖 Data Center Hardware & Networking Glossary

A quick-reference dictionary of essential data center terminology, hardware acronyms, and networking concepts.

---

## 🧱 Hardware & Server Architecture 

* **BMC (Baseboard Management Controller):** A specialized service processor that monitors the physical state of the server (temperatures, voltages, fan speeds) independently of the main OS.
* **ECC RAM (Error-Correcting Code Memory):** Specialized system memory that can detect and correct single-bit memory errors on the fly to prevent crashes.
* **IPMI (Intelligent Platform Management Interface):** An open standard protocol used by BMCs for out-of-band management and reading system event logs (SEL).
* **POST (Power-On Self-Test):** The initial diagnostic testing sequence run by the system BIOS/UEFI upon booting up hardware components.

---

## 🌐 Fiber Optics & Networking

* **SMF (Single-Mode Fiber):** Yellow-jacketed fiber optic cable with a small core (~9 µm) designed for long-distance, high-bandwidth laser signals.
* **MMF (Multi-Mode Fiber):** Aqua or lime-green jacketed fiber cable with a larger core (50 µm) used for short-range LED/VCSEL light transmission within the rack or row.
* **SFP / QSFP (Small Form-factor Pluggable):** Hot-swappable optical transceivers used to interface networking hardware with fiber or copper cabling.
* **VFL (Visual Fault Locator):** A specialized red laser tool used to visually locate micro-bends, cable fractures, or tight kinks in optical fiber strands.
